Web Dev Showdown

Build a dynamic, full-stack application from scratch.

About the Event

This is where creativity meets code. In the Web Dev Showdown, teams will be given a theme and 24 hours to build a complete web application. You'll need to handle everything from a beautiful user interface to robust backend logic and database management. Show us what you can build!

Rules and Regulations

  • Teams can consist of 2 to 3 members.
  • A secret theme will be announced at the start of the event.
  • You can use any frameworks or libraries you like (e.g., React, Node.js, Django).
  • Final projects will be judged on creativity, functionality, design, and code quality.
  • All code must be pushed to a provided Git repository before the deadline.

Prizes

  • 1st Place: $500 Cash Prize & 1-Year Vercel Pro Plan.
  • 2nd Place: JetBrains All Products Pack & Team Hoodies.
  • 3rd Place: $100 in DigitalOcean Credits & Certificates of Distinction.
